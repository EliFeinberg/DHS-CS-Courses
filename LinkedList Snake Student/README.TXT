PROJECT TITLE: Summer Snake Final Project
PURPOSE OF PROJECT: A culmination of all the CompSci skills we've aquired this year!
VERSION or DATE: 5/21/18
HOW TO START THIS PROJECT: Run the SnakeApp class
AUTHORS: Allison Wexler and Katharine Stecher
USER INSTRUCTIONS: Use the arrow keys to move the snake towards to salad and away from the pizza
	Don't hit yourself or any wall
	If you eat the salad, you get smaller
	If you eat the pizza, you get larger
	You win by getting rid of all extra squares and having a 1-square snake
